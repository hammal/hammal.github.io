cbadc

the control-bounded analog-to-digital conversion toolbox

To complement the mainly theoretical findings of my PhD thesis, I have released a python software package called cbadc. This software was intended to provide the means of doing high level analysis and simulation as well as computing the filter coefficients for control-bounded A/D converters.

The software package has a comprehensive documentation where also the installation instructions can be found. The project is an open source project and the source code is hosted on github.

Since its release, many additional features have been added as this have proven a very useful tool (as a golden model) when developing lower level circuit implementations. These and future updates are mainly announced and documented in the documentation webpage.

Bug reports, code contributions, and feature requests are most welcome and handled directly via the github webpage.
